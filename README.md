# Paradigmas
### Este repositório foi criado para armazenar códigos em diferentes paradigmas de programação como
- Paradigma imperativo (C, Fortran, Cobol)
- Paradigma Orientado a Objetos (Java, C++, C#)
- Paradigma Funcional (Haskell)
- Paradigma Lógico (Prolog)
- Paradigma concorrente (Golang)
